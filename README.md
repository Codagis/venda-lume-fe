# Commo FE

Frontend React com JavaScript e Vite.

## Pré-requisitos

- Node.js 18+
- npm ou yarn

## Instalação

```bash
npm install
```

## Desenvolvimento

```bash
npm run dev
```

Acesse http://localhost:5173 no navegador.

## Build

```bash
npm run build
```

## Preview (produção local)

```bash
npm run preview
```
