# Mapa de entregas

O mapa de entregas usa **Leaflet + OpenStreetMap** (100% gratuito). Veja [MAPA_ENTREGAS.md](./MAPA_ENTREGAS.md) para detalhes.

Os endpoints antigos do Google Maps (`map-embed-url`, `map-metrics`) permanecem disponíveis para compatibilidade, mas o frontend utiliza apenas o endpoint OSM (`map-osm`).
